# Open3D Aligner GUI

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

### 🔍 About this repo

An interactive graphical user interface (GUI) for aligning and registering 3D point clouds using the **Open3D** library in Python. This tool allows users to easily load, visualize, and align multiple point clouds in a seamless workflow.

The program supports common registration pipelines, including coarse initial alignment and fine-tuning with **ICP (Iterative Closest Point)**, making it ideal for applications in 3D scanning, robotics, and computer vision. Designed with usability in mind, the GUI simplifies complex alignment tasks while still providing flexibility for advanced users.

This repository is co-developed with **ChatGPT**, leveraging AI-assisted design to optimize pipelines, streamline implementation, and enhance the overall user experience.
